# 暗杀小游戏 - 游戏规则配置表

## 基础设置
| 设置项 | 数值 | 说明 |
|--------|------|------|
| 初始分数 | 100 | 游戏开始时的分数 |
| 错误惩罚 | 50 | 做错选择扣减的分数 |
| 时间限制 | 30秒 | 每关的限时 |
| 游戏结束 | 0分 | 分数归零时游戏结束 |

## 胜利条件详解

| 条件ID | 描述 | 触发条件 | 奖励 |
|--------|------|----------|------|
| kill_king | 杀死国王 | 国王在场 + 骑士不在场 | +10金币 |
| kill_prince | 杀死王子 | 王子在场 + 骑士不在场 | +8金币 |
| kill_pope | 杀死教皇 | 教皇在场 | +15金币 |
| collect_gold | 收集金币 | 金币道具在场 | +20金币 |
| skip_no_target | 无目标跳过 | 无任何目标在场 | +5金币 |
| skip_all_guarded | 跳过被保护场景 | 所有目标都被骑士保护 | +5金币 |

---

## 失败条件详解

| 条件ID | 描述 | 触发行为 | 惩罚 |
|--------|------|----------|------|
| kill_guarded | 刺杀被保护目标 | 刺杀国王/王子时骑士在场 | -50分 |
| kill_civilian | 杀死平民 | 点击杀死平民 | -50分 |
| attack_knight | 攻击骑士 | 尝试杀死骑士 | -30分 |
| wrong_skip | 错误跳过 | 有可刺杀目标时选择跳过 | -20分 |
| timeout | 超时 | 时间耗尽未行动 | -50分 |

---

## 场景生成规则

每关随机生成场景，包含：

1. **背景**: 从12张地图中随机选择1张
2. **目标**: 0-2个可刺杀目标
3. **守护者**: 0-2个守护者（骑士或平民）
4. **道具**: 0-2个道具

---

## 游戏流程示例

### 示例1: 国王无保护
```
场景: 国王在场，骑士不在场
正确操作: 点击"刺杀国王"
结果: +10金币，进入下一关
```

### 示例2: 国王被骑士保护
```
场景: 国王在场，骑士在场
正确操作: 点击"跳过场景"
错误操作: 点击"刺杀国王"
结果: 正确+5金币，错误-50分
```

### 示例3: 无目标场景
```
场景: 无国王/王子/教皇，只有平民
正确操作: 点击"跳过场景"
结果: +5金币，进入下一关
```

### 示例4: 教皇场景
```
场景: 教皇在场（教皇不被骑士保护）
正确操作: 点击"刺杀教皇"
结果: +15金币，进入下一关
```

## 扩展配置

如需添加新角色或道具，请修改 `config.js` 文件：

### 添加新目标
```javascript
{ id: "queen", name: "王后", image: "../res/王后.png", gold: 12 }
```

### 添加新守护者
```javascript
{ id: "guard", name: "卫兵", image: "../res/卫兵.png", blocks: ["king", "prince", "pope"] }
```

### 添加新道具
```javascript
{ id: "potion", name: "药水", image: "../res/药水.png", effect: "heal", bonus: 10 }
```
